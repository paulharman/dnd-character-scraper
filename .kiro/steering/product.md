# Product Overview

D&D Beyond Character Scraper is a comprehensive tool for extracting and parsing D&D Beyond character data into formatted markdown suitable for Obsidian and other note-taking applications.

## Core Features

- **Character Data Scraping**: Extracts complete character information from D&D Beyond API
- **Markdown Generation**: Creates well-formatted character sheets in Obsidian-compatible markdown
- **Discord Integration**: Provides notifications when character data changes
- **Rule Version Support**: Compatible with both D&D 2014 and 2024 rules
- **Spell Enhancement**: Enhanced spell data with detailed descriptions from local files
- **Customizable Output**: Configurable section ordering and formatting preferences

## Key Use Cases

- Automated character sheet generation for tabletop RPG players
- Character change monitoring and notifications via Discord
- Integration with Obsidian for campaign management and note-taking
- Backup and archival of D&D Beyond character data

## Target Users

- D&D players who use D&D Beyond for character management
- Dungeon Masters managing multiple characters and campaigns
- Users of Obsidian or similar markdown-based note-taking systems
- Groups wanting automated character change notifications
