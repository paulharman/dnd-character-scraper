# Data Directory

This directory contains all data files used for validation and testing.

## Structure
- `baseline/`: Baseline data from v5.2.0 scraper for regression testing
  - `raw/`: Raw D&D Beyond JSON API responses
  - `scraper/`: Processed JSON from v5.2.0 scraper
  - `parser/`: Markdown output from v5.2.0 parser
- `validation/`: Validation templates and test data
